04 The five interface types

What NEXUS
actually evaluates.

Every project gets assessed across five fundamentally different ways humans interact with technology.
Not all of them are right for your context.

Traditional Screen
Mobile · Web · Desktop

The most established interface type. Works for complex, frequent tasks where users need direct control and visual feedback.

Figma, Notion, any banking app
Voice Interface
Conversational · Hands-free

Natural language interaction. Works when users are moving, multitasking, or when typing creates friction.

Siri, Alexa, voice navigation
Invisible Automation
Ambient · Zero UI

The interface disappears. The system acts on your behalf. Works for highly predictable, repetitive tasks with low error consequence.

Nest thermostat, Spotify recommendations
AI Vectorial
Semantic · Generative

Understands meaning, not just commands. Works for open-ended exploration, unstructured information, and tasks without a fixed workflow.

ChatGPT, Perplexity, Notion AI
Spatial Computing
AR · VR · Mixed Reality

Three-dimensional interaction. Only justified when the task genuinely requires spatial awareness. High cost, high complexity, narrow fit.

Apple Vision Pro, HoloLens, IKEA Place

User values priority

The highest weighted factor. Even technically perfect products fail when they contradict organizational or user values. Google Glass failed not because the tech was wrong, but because it violated social values around privacy and acceptance. Values are assessed first to filter interface types that are fundamentally misaligned before any technical scoring begins.

08 Why open source

The framework belongs
to everyone who uses it.

We built NEXUS to fix a real problem.
Keeping it behind a paywall would contradict everything the tool stands for.

01

Better decisions shouldn't cost money

Interface decisions affect everyone who uses the product. The framework that informs those decisions should be accessible to a solo designer, a startup, and a corporation equally.

02

Transparency builds trust

NEXUS guides your decision, it does not make it for you. Opening the methodology means you can inspect every weight, every factor, every assumption. No black box.

03

The tool improves when more people use it

Every project assessed adds signal. Patterns emerge. The framework gets sharper. Open source is not charity. It is the fastest way to build something genuinely useful.
